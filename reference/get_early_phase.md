# Get early phase temporal component (density)

Get early phase temporal component (density)

## Usage

``` r
get_early_phase(time, thalf, eta = 1, gamma = 0)
```

## Arguments

- time:

  Numeric vector of time values.

- thalf:

  Half-life parameter.

- eta:

  Exponent of time (nu in decompos notation). Default 1.

- gamma:

  Shape parameter (m in decompos notation). Default 0.

## Value

Numeric vector of early phase density values.
