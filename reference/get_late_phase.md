# Get late phase temporal component (hazard)

Get late phase temporal component (hazard)

## Usage

``` r
get_late_phase(time, thalf, eta, gamma = 0)
```

## Arguments

- time:

  Numeric vector of time values.

- thalf:

  Half-life parameter.

- eta:

  Exponent of time (nu in decompos notation).

- gamma:

  Shape parameter (m in decompos notation). Default 0.

## Value

Numeric vector of late phase hazard values.
